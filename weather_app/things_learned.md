# Takeaways

- Using `vite` for dev server
- Best practices for vanilla JS

HTML
- `Template` element
- `data-` attribute
- Various shortcuts

CSS
- `blur` + `overflow hidden` "restricted" effect
- Nested grid, flex, etc.
- `nth-child`, `>` tags

JS
- API calling with `Axios`
    - [open-meteo](https://open-meteo.com/en/docs) weather API
    - GeoLocation API
    - Reading documentation
- Parsing objects
    - Object deconstruction
- Arrow function syntax
    - Without braces, return is implied
    - With braces, return is required (don't always need to return something)
- Array methods: `map`, `filter`, `forEach`, etc.
- `Promise` handling
    - `.then`, `.catch`
- Callbacks
- Import/export syntax, modularity
- Working with DOM
    - Generating dynamic content with templates
    - `querySelector`, `getElementById`, etc.
